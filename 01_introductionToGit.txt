# What is gitHub ??
GitHub is a cloud platform that hosts Git repositories. / GitHub = Cloud storage + collaboration for Git repositories


Think like this:
  Git = tool
  GitHub = service/platform


Other Git platforms:
  GitHub (most popular)
  GitLab
  Bitbucket

Why GitHub?
  Remote backup of code
  Collaboration
  Pull Requests (PR)
  Code review
  CI/CD
  Issues, Projects, Wikis



Git vs GitHub

| Git               | GitHub             |
| ----------------- | ------------------ |
| Installed locally | Cloud platform     |
| Tracks code       | Hosts repositories |
| Works offline     | Needs internet     |
| CLI based         | Web + CLI          |




You can use Git without GitHub
You cannot use GitHub without Git
